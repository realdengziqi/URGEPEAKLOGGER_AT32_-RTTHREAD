# 罗氏线圈工作原理和测量方法

本文说明罗氏线圈的基本原理、浪涌电流测量方法，以及本项目在固件和硬件设计中需要注意的事项。

当前项目状态：外部罗氏线圈已确认没有积分器，线圈具体参数暂时待后续标定。因此本项目真实浪涌采集路线先按“高速 ADC 采样 + 数字积分”规划；线圈参数缺失不阻塞采样框架设计，但会影响最终 kA 换算和精度。

## 罗氏线圈是什么

罗氏线圈是一种用于测量交流电流、脉冲电流和浪涌电流的空心线圈。它通常绕在被测导体周围，不需要铁芯。

被测电流穿过线圈中心时，会在线圈中感应出电压。这个感应电压不是直接与电流成正比，而是与电流变化率成正比。

简化关系为：

```text
v_out = M * di/dt
```

其中：

- `v_out` 是罗氏线圈输出电压。
- `M` 是线圈互感系数。
- `i` 是被测电流。
- `di/dt` 是电流变化率。

因此，原始罗氏线圈天然测到的是“电流变化有多快”，不是“电流有多大”。

## 为什么罗氏线圈需要积分

浪涌记录仪关心的是电流峰值，例如：

```text
0.5kA
1kA
5kA
20kA
100kA
```

但原始罗氏线圈输出的是：

```text
di/dt
```

要从 `di/dt` 得到电流 `i`，必须做积分：

```text
i = integral(di/dt) dt
```

工程上有两种常见方法：

| 方法 | 说明 |
| --- | --- |
| 模拟积分 | 用运放、电阻、电容组成积分器，把罗氏线圈输出转换成与电流近似成正比的电压。 |
| 数字积分 | MCU 或采样系统高速采集原始波形，再用软件做积分。 |

如果没有积分，直接取罗氏线圈输出峰值，得到的是最大电流变化率，不是真实电流峰值。

## 原始线圈和带积分器传感器的区别

### 原始罗氏线圈

原始罗氏线圈输出关系为：

```text
v_out = M * di/dt
```

这种线圈需要外部积分器或数字积分。本项目当前外部线圈属于这一类，并且没有外部积分器。

需要确认的参数包括：

- 互感系数 `M`。
- 输出单位，例如 `V/(A/s)` 或 `mV/(kA/us)`。
- 线圈带宽。
- 输出极性。
- 线圈和线缆等效参数。
- 推荐负载阻抗。

### 带积分器传感器

带积分器传感器内部已经把 `di/dt` 积分成电流相关信号。

输出关系通常可近似为：

```text
v_out = K * current
```

例如：

```text
1kA -> 10mV
```

这种情况下可以按比例系数换算电流：

```text
current_kA = output_voltage_mV / 10mV
```

但前提是必须确认该比例系数的标定条件。

本项目当前不按带积分器传感器设计第一版真实浪涌采集。

## 浪涌波形

防雷和 SPD 场景常见浪涌电流波形包括：

| 波形 | 常见用途 | 特点 |
| --- | --- | --- |
| `8/20us` | T2/T3 浪涌测试 | 上升快，持续时间短。 |
| `10/350us` | T1 雷击电流测试 | 能量更大，持续时间更长。 |

竞品资料中也区分了：

```text
T1 使用 10/350us 线圈
T2/T3 使用 8/20us 线圈
```

这意味着线圈、前端带宽、采样窗口和算法不能简单混用。

## 需要测量哪些量

浪涌峰值记录仪通常不只记录峰值，还可能记录：

- 浪涌次数。
- 电流峰值。
- 极性。
- 波前时间。
- 半峰值时间。
- 电流陡度。
- Q 值或电荷量。
- 能量。
- 多脉冲间隔。
- 发生日期和时间。

这些量对采样能力要求不同。

只做触发计数，要求最低。  
要做峰值记录，需要准确捕捉峰值。  
要做波前时间、半峰值时间、能量和 Q 值，则需要保存或重建较完整的波形。

## 峰值测量方法

如果传感器输出已经与电流成正比，可以直接寻找波形峰值：

```text
peak_current = peak_voltage / sensitivity
```

本项目外部线圈已确认是无积分器原始罗氏线圈，因此不能直接找电压峰值，而应该：

1. 高速采样原始线圈输出。
2. 去除零点偏置。
3. 按采样周期积分。
4. 得到电流波形。
5. 在电流波形上寻找峰值。

数字积分简化形式：

```text
current[n] = current[n-1] + gain * (sample[n] - baseline) * sample_period
```

这里的 `gain` 取决于：

- 罗氏线圈互感。
- 模拟前端增益。
- ADC 参考电压。
- ADC 分辨率。
- 输入偏置。
- 单位换算。

在线圈参数未确认前，可以先把 `gain` 作为待标定系数，优先实现稳定的高速采样、基线扣除、触发、积分窗口和 raw 波形诊断。

## 触发方法

浪涌触发可以有几种方式：

| 方法 | 说明 | 优点 | 风险 |
| --- | --- | --- | --- |
| ADC 轮询 | 周期读取 ADC 判断阈值 | 简单 | 容易漏掉快速浪涌。 |
| 定时器触发 ADC | 用固定采样率采样 | 可控 | 需要足够高采样率和缓冲。 |
| ADC DMA 环形缓冲 | 持续高速采样并保留预触发数据 | 适合捕获瞬态 | 占 RAM 和 CPU。 |
| 模拟比较器触发 | 硬件先判断超过阈值 | 响应快 | 需要模拟阈值和防误触发设计。 |
| 外部触发 + ADC 捕获 | 比较器或专用前端触发 MCU | 响应快 | 硬件复杂度更高。 |

本项目当前每秒读取一次 PC1 raw，只适合验证后端数据通路，不适合真实浪涌捕获。

## 峰值保持窗口

峰值保持窗口必须覆盖完整波形。

建议初步原则：

| 波形 | 建议窗口 |
| --- | --- |
| `8/20us` | 至少大于 `100us`，具体取决于采样率和前端带宽。 |
| `10/350us` | 至少大于 `1~2ms`，具体取决于线圈和积分器响应。 |

如果还要记录多脉冲，应在一次事件后继续观察一段时间。竞品资料中提到最多记录 `20` 个连续脉冲，并提到脉冲间隔 `>=40ms`。

## 极性判断

罗氏线圈有方向性。

同一个浪涌电流，如果线圈安装方向反了，输出极性也会反。

因此必须确认：

- 线圈外壳上是否有方向箭头。
- 正向电流穿过线圈时输出是正脉冲还是负脉冲。
- 本板 ADC 偏置后，高于 `1.65V` 代表正脉冲还是负脉冲。
- 现场安装方向是否固定。

如果不确认极性，记录中的“正浪涌/负浪涌”没有可靠意义。

## 比例系数标定

线圈比例系数不能只看一个标签，还要绑定标定条件。

例如 `1kA/10mV` 需要确认：

- 是否表示 `1kA -> 10mV`。
- 是原始线圈输出，还是积分后输出。
- 对应 `8/20us` 还是 `10/350us`。
- 标定时线缆长度是多少。
- 标定时负载阻抗是多少。
- 输出端是开路、50 欧负载，还是接入特定仪表。
- 误差范围是多少。

如果这些条件不同，实际比例系数可能不同。

## 采样率建议

真实采样率需要结合目标波形和算法确定。

基本原则：

- 捕获 `8/20us` 上升沿时，采样率不能太低。
- 要计算波前时间和电流陡度，需要比只抓峰值更高的采样率。
- 要做数字积分，采样周期必须稳定。
- ADC DMA 比线程里轮询 ADC 更适合捕获浪涌瞬态。

当前阶段建议先不要在固件里写死最终采样率，而是先通过示波器确认前端实际波形宽度和幅度。

## 本项目当前建议路线

在传感器参数没有完全确认前，建议分阶段推进。外部线圈无积分器这一点已经确认，所以固件路线不再等待“是否积分”的结论。

第一阶段：通路验证

- 保留 PC1 raw 读取。
- 保留 `baseline`、`delta raw`、触发状态、触发计数。
- Modbus 和 MSH 只展示调试 raw 数据。

第二阶段：模拟前端实测

- 用信号源从 SMA 输入已知脉冲。
- 测 `RF1`、运放输出、PC1 ADC 节点。
- 确认偏置、电压范围、实际增益和带宽。

第三阶段：传感器标定

- 使用标准浪涌源或供应商标定报告。
- 确认 `kA -> mV` 或 `di/dt -> V` 的真实关系。
- 确认线缆长度和负载影响。

第四阶段：固件算法

- 固件按无积分器原始罗氏线圈处理，使用高速 ADC 采样和数字积分。
- 线圈互感或比例系数未知时，先输出 raw、积分相对量和待标定状态，不输出正式 kA。
- 正式 Modbus 寄存器再输出 kA 峰值、极性、事件时间和记录字段。

## 结论

罗氏线圈本身测的是 `di/dt`，不是直接测电流峰值。要得到真实 kA，必须有积分过程。

外部传感器已确认没有积分器，而本板前端也没有看到明确模拟积分器结构，因此固件必须通过高速采样和数字积分来计算电流，否则只能记录 raw 峰值和触发状态。

在参数未确认前，本项目不能把 PC1 ADC raw 值解释成真实 kA，只能作为调试量使用。
