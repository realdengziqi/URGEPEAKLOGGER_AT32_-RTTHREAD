# 浪涌输入运放前端工作逻辑分析

本文说明主控板浪涌输入模拟前端的工作逻辑。分析依据为：

- `hardware_info/主控板_BOM.csv`
- `hardware_info/主控板_网表.csv`

本文只基于当前 BOM 和网表推断，不等同于经过示波器实测确认的最终模拟参数。

## 相关器件

| 器件 | 型号/网络 | 作用 |
| --- | --- | --- |
| `RF1` | SMA 接口 | 外部浪涌传感器输入接口，中心针接信号，外壳接 `AGND`。 |
| `U20` | `TLV2372IDR` | 双运放，用于输入缓冲和 ADC 偏置缓冲。 |
| `U5` | `TP7660` | 电荷泵，生成运放负电源。 |
| `OP_+5V` | 运放正电源 | 给 `U20` 供正电源。 |
| `OP_-5V` | 运放负电源 | 给 `U20` 供负电源。 |
| `MCU_PC1_ADC2_IN11` | MCU ADC 输入 | 浪涌采样 ADC 通道。 |

## 信号链路

从网表看，浪涌输入信号链路大致如下：

```text
RF1 SMA 中心针
  -> R41 22k
  -> 输入分压/保护节点 $7N4
  -> U20A 第一路运放跟随器
  -> R42 1k
  -> 偏置节点 $7N13
  -> U20B 第二路运放跟随器
  -> R39 51R
  -> ADC 输入节点
  -> R40 0R
  -> MCU_PC1_ADC2_IN11
```

输入节点 `$7N4` 还连接：

```text
R43 3.4k -> AGND
C51 33pF -> AGND
D3 -> OP_+5V / OP_-5V 钳位
```

偏置节点 `$7N13` 还连接：

```text
R38 1k -> 3V3A
```

ADC 输入节点还连接：

```text
C50 1nF -> AGND
```

## 第一级：输入衰减和保护

SMA 中心针先经过 `R41=22k` 到输入节点 `$7N4`，该节点再通过 `R43=3.4k` 到 `AGND`。

因此输入信号被分压：

```text
分压比例 = 3.4k / (22k + 3.4k)
         ~= 0.134
```

也就是说，SMA 端输入 `1V`，到 `$7N4` 大约只剩 `134mV`。

这一部分的作用是：

- 降低外部传感器输入幅度。
- 限制进入运放输入脚的电流。
- 配合 `D3` 做过压钳位保护。
- 配合 `C51=33pF` 抑制很高频的尖峰。

`D3` 接到 `OP_+5V / OP_-5V`，用于把输入节点限制在运放供电范围附近，避免浪涌传感器的尖峰直接打到运放输入。

## 第二级：第一路运放缓冲

`U20A` 第一路运放的连接方式为：

```text
U20 pin1 输出 -> U20 pin2 反相输入
U20 pin3 同相输入 <- $7N4
```

这是典型电压跟随器。

它的作用不是放大，而是缓冲：

- 前级分压和保护网络不用直接驱动后级。
- 后级偏置网络不会明显改变输入分压比例。
- 提高模拟链路输入节点的稳定性。

## 第三级：把双极性信号抬到 ADC 中点

第一路运放输出经过 `R42=1k` 到 `$7N13`。同时，`3V3A` 经过 `R38=1k` 也接到 `$7N13`。

因为两个电阻都是 `1k`，所以 `$7N13` 近似为第一路输出和 `3V3A` 的平均值：

```text
V_$7N13 = (V_U20A + 3.3V) / 2
```

如果没有输入信号，第一路输出约为 `0V`，则：

```text
V_$7N13 ~= 1.65V
```

这一步的作用是把原本围绕 `0V` 上下摆动的双极性脉冲，搬移到 ADC 可采样范围中间。

因此 ADC 看到的信号不是围绕 `0V`，而是围绕 `1.65V`：

```text
正脉冲：ADC 电压高于 1.65V
负脉冲：ADC 电压低于 1.65V
无信号：ADC 电压约为 1.65V
```

同时，因为是两个 `1k` 平均，交流信号幅度又被乘以 `0.5`。

## 第四级：第二路运放缓冲

`U20B` 第二路运放的连接方式为：

```text
U20 pin7 输出 -> U20 pin6 反相输入
U20 pin5 同相输入 <- $7N13
```

这也是电压跟随器。

它的作用是：

- 把 `1.65V` 偏置后的信号低阻输出。
- 驱动 MCU ADC 的采样电容。
- 减少 ADC 采样瞬间对前级偏置网络的扰动。

## 第五级：ADC 前滤波和隔离

第二路运放输出经过 `R39=51R` 到 ADC 输入节点，节点上有 `C50=1nF` 到 `AGND`，再通过 `R40=0R` 到 `MCU_PC1_ADC2_IN11`。

`R39` 和 `C50` 形成一个小 RC 滤波：

```text
时间常数 = 51R * 1nF = 51ns
截止频率约为 1 / (2 * pi * 51ns) ~= 3.1MHz
```

这一部分的作用是：

- 滤掉很高频的尖峰。
- 隔离运放输出和 ADC 采样电容。
- 改善 ADC 采样稳定性。

## 总体传递关系

按小信号近似计算：

```text
第一级输入分压：0.134
偏置平均网络：0.5
总交流增益：0.134 * 0.5 ~= 0.067
```

所以 ADC 输入电压大致为：

```text
V_ADC ~= 1.65V + V_SMA * 0.067
```

例子：

```text
SMA 输入 +1V  -> ADC 约 1.65V + 67mV
SMA 输入 -1V  -> ADC 约 1.65V - 67mV
SMA 输入  0V  -> ADC 约 1.65V
```

## 这不是罗氏线圈积分器

当前网表没有看到典型罗氏线圈积分器结构。

典型模拟积分器通常会有：

```text
输入电阻 -> 运放反相输入
反馈电容 -> 运放输出到反相输入
反馈电阻 -> 与反馈电容并联，用于低频泄放和防止直流饱和
```

而当前电路更像：

```text
输入保护
  -> 衰减
  -> 缓冲
  -> 加 1.65V ADC 偏置
  -> 再缓冲
  -> ADC 滤波
```

目前外部传感器已确认是无积分器原始罗氏线圈，因此当前板上运放前端大概率没有把 `di/dt` 转换成真实电流波形。

在这种情况下：

- ADC 峰值不能直接等于电流峰值。
- 固件不能直接把 raw 峰值换算成 kA。
- 若要得到电流峰值，需要数字积分，或后续硬件改版增加模拟积分器。

## 对 1kA/10mV 标识的影响

如果外部线圈的 `1kA/10mV` 表示 SMA 端输出灵敏度为：

```text
1kA -> 10mV
```

那么经过本板前端后：

```text
ADC 端信号幅度 ~= 10mV/kA * 0.067
              ~= 0.67mV/kA
```

12 位 ADC、3.3V 参考时：

```text
1 LSB ~= 3.3V / 4096 ~= 0.805mV
```

也就是说：

```text
1kA 约等于 0.83 个 ADC count
```

这个量级太小，不适合可靠做 `0.5kA` 触发或 `0.1kA` 分辨率测量。

这说明至少有一个条件需要重新确认：

- `1kA/10mV` 不是 SMA 端直接输出灵敏度。
- 该线圈不是当前板子原计划匹配的传感器。
- 当前前端原本面向更高幅度的罗氏线圈原始脉冲。
- 当前硬件需要增加模拟积分和增益。
- 固件路线应确定为高速采样后做数字积分，但前端噪声和 ADC 分辨率仍需验证。

## 建议示波器测试点

建议用示波器同时观察以下节点：

| 测试点 | 目的 |
| --- | --- |
| `RF1` SMA 中心针 | 确认外部传感器实际输出幅度、极性和波形。 |
| `$7N4` | 确认输入分压、钳位和高频尖峰情况。 |
| `U20A` 输出 | 确认第一路缓冲后波形。 |
| `$7N13` | 确认 1.65V 偏置和幅度减半是否符合预期。 |
| `U20B` 输出 | 确认送 ADC 前的最终模拟波形。 |
| `MCU_PC1_ADC2_IN11` | 确认 ADC 实际采样节点波形。 |

测试时建议先用信号源替代线圈输入小幅正负脉冲，确认前端传递关系，再接真实线圈做浪涌或等效脉冲测试。
